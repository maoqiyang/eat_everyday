# Eat Everyday - 每日菜单规划助手

一个精美的网页应用，帮助用户自定义每日菜单并实现随机菜品抽取功能。

## 功能特点

### 核心功能
- **菜品管理**：添加、删除、编辑菜品，支持设置名称、分类、难度等级和抽取权重
- **随机抽取**：智能随机算法，支持排除最近N天已选菜品，可自定义每餐菜品数量
- **历史记录**：自动保存每日抽取结果，支持按日期和分类查询

### 视觉与交互设计
- **响应式布局**：完美适配移动端、平板和桌面设备
- **动态视觉效果**：抽取过程动画、卡片式设计、悬停反馈
- **现代化UI**：基于Tailwind CSS的精美界面，符合餐饮主题的配色方案

### 数据持久化
- **本地存储**：使用localStorage保存用户数据
- **数据导入/导出**：支持JSON格式的数据备份与恢复

### 增强功能
- **权重设置**：为特定菜品设置更高的抽取概率
- **分享功能**：生成并分享菜品结果
- **多语言支持**：支持中英文界面切换

## 技术栈
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Font Awesome

## 快速开始

### 本地运行
1. 克隆本仓库
```bash
git clone https://github.com/your-username/eat_everyday.git
```

2. 进入项目目录
```bash
cd eat_everyday
```

3. 使用浏览器打开 `index.html` 文件即可开始使用

### 部署方法
项目已配置CI/CD流程，推送到main分支后会自动部署到服务器。

## 使用说明

### 添加菜品
1. 点击"管理菜单"进入菜品管理页面
2. 填写菜品信息（名称、分类、难度等）
3. 点击"添加菜品"按钮完成添加

### 随机抽取
1. 点击"随机抽取"进入抽取页面
2. 设置抽取条件（排除天数、菜品数量、选择分类）
3. 点击"开始抽取"或"重新抽取"按钮生成随机菜单
4. 可选择保存结果或分享结果

### 查看历史
1. 点击"历史记录"进入历史页面
2. 可使用筛选器按日期或分类查询历史记录

### 数据管理
1. 在菜品管理页面，可点击"导出数据"备份所有数据
2. 点击"导入数据"可恢复之前备份的数据

## 开发说明
- 项目使用模块化的JavaScript设计，核心逻辑位于`app.js`文件
- 样式使用Tailwind CSS实现，自定义配置位于HTML文件的`<script>`标签中
- 所有用户数据存储在浏览器的localStorage中
